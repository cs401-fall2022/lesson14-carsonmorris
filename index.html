Learn Typescript: For Javascript Programmers

Introduction

This page will cover the basics of type script and its corresponding relationship with Javascript. Typescript and Javascript have an unusual relationship, where Javascript code is already Typescript code. For each section below, we will cover some of the basics of Typescript.

Types in Typescript

In Typescript, you can indicate a variable's type based on what you assign it to. Instead of declaring that myString is a string, you can just set it equal to a string. See below:

let myString = "Hello cruel and evil world.";
